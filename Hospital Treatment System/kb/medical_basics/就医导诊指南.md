# 常见症状就医导诊指南

## 呼吸系统症状

### 咳嗽/咳痰
- **伴随发热、胸痛** → 呼吸内科
- **慢性咳嗽（>8周）** → 呼吸内科
- **咳嗽伴鼻塞流涕** → 呼吸内科 或 耳鼻喉科

### 发热
- **低热（<38.5℃）伴普通感冒症状** → 呼吸内科 或 发热门诊
- **高热（≥38.5℃）持续不退** → 发热门诊 → 根据诊断转相应科室
- **发热伴皮疹** → 感染科

### 呼吸困难
- **急性呼吸困难** → 急诊科
- **慢性呼吸困难** → 呼吸内科 或 心内科

## 消化系统症状

### 腹痛
- **右上腹痛** → 肝胆外科/消化内科
- **右下腹痛** → 普外科（警惕阑尾炎）
- **上腹痛伴反酸** → 消化内科
- **全腹痛/剧烈腹痛** → 急诊科

### 腹泻/呕吐
- **急性（<2周）** → 消化内科 或 肠道门诊
- **慢性腹泻** → 消化内科

## 心脑血管症状

### 胸痛
- **剧烈压榨样胸痛，放射至左肩/下巴** → 急诊科（警惕心梗）
- **胸闷气短** → 心内科
- **胸痛与呼吸相关** → 呼吸内科

### 头晕/头痛
- **持续性头痛** → 神经内科
- **眩晕（天旋地转）** → 耳鼻喉科 或 神经内科
- **头痛伴视力改变** → 眼科 或 神经内科

## 其他常见症状

### 皮肤问题
- **皮疹/过敏** → 皮肤科
- **皮肤外伤** → 急诊科/普外科

### 腰背痛
- **急性腰扭伤** → 骨科
- **慢性腰痛** → 骨科 或 康复科
- **腰痛伴下肢放射痛** → 骨科（警惕椎间盘突出）

## 儿科建议
- 14岁以下儿童首选**儿科**
- 夜间突发高热/抽搐 → 儿科急诊

## 妇产科建议
- 孕期不适 → 产科
- 妇科炎症/月经问题 → 妇科

## 急诊指征（需要立即就医）
- 剧烈胸痛/胸闷
- 严重呼吸困难
- 大出血
- 意识丧失/抽搐
- 严重外伤
- 过敏反应（呼吸困难、喉头水肿）
